> ZERO dependencies

## Installation

```shell
$ npm i --save-dev @amazingandyyy/git-hooks
# or
$ yarn add --dev @amazingandyyy/git-hooks
```

## Config

package.json

```json
{
   "git-hooks": {
      "precommit": "npm run test",
      "prepush": "npm run test"
   }
}
```
